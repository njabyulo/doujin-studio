## What I've Built

### Core Features (Implemented)
- **AI Script Generation**: URL-to-ad conversion using Google Gemini AI
- **30-Second Video Ads**: Programmatic video rendering with Remotion
- **Real-Time Preview**: Interactive video player with scene-by-scene editing
- **Full Customization**: Edit text overlays, colors, fonts, timing, and scene duration
- **Professional Output**: Export to MP4 with customizable branding

### Technical Stack (Cost-Effective)
- **AI**: Google Gemini 2.5 Flash (affordable, high-quality)
- **Video Rendering**: Remotion (open-source, React-based)
- **Frontend**: Next.js 16 (modern, fast, SEO-friendly)
- **Backend**: Serverless functions (pay-per-use, scales automatically)
- **Infrastructure**: AWS Lambda + S3 (cost-effective at scale)

---

## How It Works

1. **Input**: User provides a URL (website, product page, etc.)
2. **AI Analysis**: Gemini AI analyzes content and generates compelling ad script with 3-6 scenes
3. **Customization**: User edits text, colors, fonts, timing in real-time preview
4. **Rendering**: Remotion generates professional 30-second video
5. **Export**: Download MP4 ready for social media, ads, etc.
